# 🔐 Password Strength Analyzer

A full-stack cybersecurity web application that analyzes password strength based on security rules like length, complexity, uniqueness, and password reuse prevention.

## 📌 Features

✅ Password strength analysis  
✅ Checks uppercase and lowercase letters  
✅ Checks numbers and special characters  
✅ Detects weak/common passwords  
✅ Suggests stronger passwords  
✅ Password hashing using SHA-256  
✅ Password reuse prevention using SQLite  
✅ Responsive modern UI  
✅ Full-stack Flask web application

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/dhanyasreegopinigari-blue/password-strength-analyzer.git
```

---

### 2️⃣ Open Project Folder

```bash
cd password-strength-analyzer
```

---

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4️⃣ Run Flask Server

```bash
python app.py
```

---

### 5️⃣ Open in Browser

```text
http://127.0.0.1:5000
```
